HTMLとCSSの公開について

①ファイルを作成する

まず、自分のパソコン上でWebページを構成するファイルを作成する。
・index.html：Webページの本体
・style.css：ページのデザインを記述するスタイルシート

※今回はDreamweaverを使って、実際にコードを書きながら作成している。

②Gitでファイルを管理する

作成したファイルを「変更履歴付き」で管理するために、Gitというツールを用いる。
作業フォルダ内でGitBashを使って以下のコマンドを実行する：

git init：Git管理を開始する

git add .：すべてのファイルをステージング

git commit -m "初回コミット"：初期状態として履歴に保存

・その後も修正のたびに、以下のように段階的にコミットしていく。

git add .

git commit -m "自己紹介セクションを追加"

③GitHubにファイルをアップロードする

GitHub上にリポジトリを作成する。
GitBashで以下のようにGitHubと接続し、ファイルをアップロードする：

git remote add origin https://github.com/ユーザー名/リポジトリ名.git

git branch -M main：mainブランチに切り替え

git push -u origin main：GitHubに反映

4.GitHub PagesでWeb公開する

GitHubには「Pages」という機能があり、アップロードしたHTMLをWebページとしてそのまま公開できる。

リポジトリの「Settings」→「Pages」に移動し、

・Branch：main

・Folder：/rootを選択することでHTMLページがWeb上で公開される

・数十秒〜1分でURLが発行され、世界中からアクセス可能になる。
